# Customer Support System - Quick Start Guide

## 🚀 Getting Started

### 1. Access Support
Click the **Headphones icon** (🎧) in the navigation bar to open the support page.

### 2. Select an Order (Optional)
- When the page loads, you'll see a modal with all your orders
- Click on any order to get order-specific support
- Or click **"Skip - Ask general question"** for non-order queries

### 3. Start Chatting
- Type your message in the text area
- Press **Enter** to send (Shift+Enter for new line)
- Upload images or videos using the buttons
- Your conversation is automatically saved

---

## 📸 Uploading Files

### Image Upload
1. Click the **Image icon** (📷) next to the text area
2. Select an image from your device
3. Wait for upload confirmation
4. Image appears in preview below chat
5. Click **X** to remove before sending

### Video Upload
1. Click the **Video icon** (🎥) next to the text area
2. Select a video from your device
3. Wait for upload confirmation
4. Video appears in preview below chat
5. Click **X** to remove before sending

## 🔧 Troubleshooting

### Message Not Sending
- Check internet connection
- Ensure you're logged in
- Try refreshing the page
- Contact admin if persists

### Upload Failed
- Check file size (max limits apply)
- Verify file format (jpg, png, mp4, etc.)
- Ensure stable connection
- Try again or use smaller file

### Chat History Not Loading
- Ensure you're using same account
- Clear browser cache if needed
- Check login status
- Try different browser

### AI Not Responding
- Wait a few seconds (processing time)
- Check for error messages
- Retry sending message
- Contact support directly if urgent
